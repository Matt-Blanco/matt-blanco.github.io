Absurd Fabrications

Project Type: Fabricated Objects

Over the course of seven weeks, five objects were fabricated from a variety of different prompts all based on the central theme of absurd. The goals were to become comfortable with a variety of fabrication techniques, and as well as play with the idea of what absurd means. Each object was constructed around a different interpretation of absurd.

The Box
Brief: Create a box from one piece of wood with a single pivot point.

A box with an additional, albeit useless feature. One lid opens to a hallow space while a second lid opens to nothing.

Chairs
Brief: Create five identical items replicated using jigs.

A collection of five tiny chairs. Too small for a person and too large for a doll-house.

Cups
Brief: Create a two identical items, one small and one large using a laser cutter.

It's assumed cups should be able to hold liquids, but what if they can't.

The One Page Holder
Brief: Create an enclosure to hold an object or a set of objects

An object with a useless function. A specially designed enclosure to fit a single sheet of 8.5"x11" paper.

A Lamp?
Brief: Create an object primarily using two different materials and various fasteners

A lamp and table combination, that when connected to an AC to DC power converter, only turns on when all the previously made objects are placed into the correct space.
